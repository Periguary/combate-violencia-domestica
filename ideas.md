# Ideias de Design: Combate à Violência Doméstica - ODS 16

## Resposta 1: Minimalismo Institucional com Propósito Social (Probabilidade: 0.08)

**Design Movement**: Minimalismo Institucional + Humanismo Digital

**Core Principles**:
- Clareza radical: cada elemento serve um propósito comunicativo direto
- Acessibilidade em primeiro lugar: tipografia legível, contraste alto, navegação intuitiva
- Humanidade através de espaço: uso generoso de whitespace para criar respiro emocional
- Confiança através de consistência: paleta restrita, hierarquia clara

**Color Philosophy**:
Paleta baseada em tons de terra e azul profundo. O azul (principal) evoca confiança institucional e segurança. Tons de terra (bege, ocre suave) humanizam a interface. Vermelho suave apenas para alertas críticos (denúncia, emergência). A filosofia é: cores que transmitem seriedade sem frieza, esperança sem ingenuidade.

**Layout Paradigm**:
Estrutura assimétrica com coluna principal de conteúdo (2/3) e painel lateral de recursos (1/3). Seções separadas por whitespace generoso em vez de bordas. Uso de grid 12-colunas com alinhamento à esquerda para criar movimento visual.

**Signature Elements**:
1. Ícones customizados em estilo line-art (não preenchidos) que reforçam profissionalismo
2. Citações ou dados estatísticos em blocos destacados com fundo suave
3. Linhas horizontais sutis como separadores visuais

**Interaction Philosophy**:
Transições suaves (300ms) em hover. Botões com estado claro (default, hover, active). Formulários com validação em tempo real e mensagens de erro empáticas. Nenhuma animação desnecessária—cada movimento tem propósito.

**Animation**:
- Fade-in suave (0.3s) para elementos ao carregar
- Hover: mudança de cor + leve elevação (box-shadow)
- Transições de página: fade out/fade in (0.2s)
- Nenhuma animação automática ou looping

**Typography System**:
- Display: Serif bold (ex: Playfair Display) para títulos principais
- Body: Sans-serif regular (ex: Source Sans Pro) para textos longos
- Hierarquia: 2.5rem (h1), 1.875rem (h2), 1.25rem (h3), 1rem (body)
- Line-height: 1.6 para corpo, 1.2 para títulos

## Resposta 2: Design Humanista com Elementos Orgânicos (Probabilidade: 0.07)

**Design Movement**: Humanismo Contemporâneo + Organic Design

**Core Principles**:
- Formas suaves e arredondadas que evocam acolhimento
- Narrativa visual que conecta emocionalmente com vítimas e apoiadores
- Diversidade representada visualmente através de ilustrações inclusivas
- Movimento natural e fluido em toda a interface

**Color Philosophy**:
Paleta aquarela com tons pastéis: rosa suave, verde menta, azul céu. Estes tons evocam calma, cura e esperança. Complementados por tons mais saturados (coral, verde-floresta) para CTAs. A filosofia é criar um espaço que se sinta seguro e acolhedor, não clínico.

**Layout Paradigm**:
Seções com formas orgânicas (bordas arredondadas, clip-paths com ângulos suaves). Alternância entre seções full-width e conteúdo centrado. Uso de ilustrações grandes como divisores visuais entre seções.

**Signature Elements**:
1. Ilustrações customizadas mostrando pessoas de diferentes origens em contextos de apoio
2. Padrões geométricos suaves (círculos, ondas) como fundo de seções
3. Ícones com preenchimento suave (não line-art)

**Interaction Philosophy**:
Interações lúdicas mas respeitosas. Hover effects que revelam mais informação (ex: cards que se expandem). Micro-interações que fazem o usuário sentir que está sendo "cuidado" pela interface.

**Animation**:
- Entrance: elementos deslizam suavemente de baixo para cima (0.5s)
- Hover: cards ganham sombra suave e se elevam levemente
- Scroll: parallax suave em imagens de fundo
- Transições de página: dissolve suave (0.3s)

**Typography System**:
- Display: Rounded sans-serif (ex: Poppins Bold) para títulos
- Body: Rounded sans-serif (ex: Poppins Regular) para textos
- Hierarquia: 2.25rem (h1), 1.75rem (h2), 1.125rem (h3), 1rem (body)
- Line-height: 1.7 para corpo (mais espaçado), 1.3 para títulos

## Resposta 3: Design Assertivo com Tipografia Forte (Probabilidade: 0.06)

**Design Movement**: Tipografia Expressiva + Modernismo Crítico

**Core Principles**:
- Tipografia como protagonista: fontes grandes, pesos variados, espaçamento dramático
- Contraste visual agressivo: preto/branco com acentos em cores saturadas
- Estrutura editorial: o site lê como um manifesto, não apenas informação
- Urgência comunicada através de design

**Color Philosophy**:
Paleta de alto contraste: preto profundo, branco puro, com acentos em vermelho vibrante e amarelo ouro. O vermelho comunica urgência e ação. O amarelo ouro traz esperança. A filosofia é: design que não pode ser ignorado, que comunica a seriedade do tema.

**Layout Paradigm**:
Assimetria radical. Títulos grandes ocupam espaço generoso. Uso de colunas desiguais (ex: 1/3 + 2/3). Blocos de cor sólida como fundo de seções. Tipografia que "quebra" a grid para criar movimento.

**Signature Elements**:
1. Tipografia oversized (até 4rem) para títulos principais
2. Blocos de cor sólida (preto, vermelho) como fundos de seções
3. Linhas diagonais ou ângulos como elementos decorativos

**Interaction Philosophy**:
Interações diretas e responsivas. Botões com mudanças de cor agressivas. Feedback imediato e óbvio. Nada de sutileza—o usuário deve sentir que está interagindo com algo importante.

**Animation**:
- Entrance: elementos aparecem com scale-up (0.3s)
- Hover: mudança de cor + scale (1.05x)
- Transições de página: slide horizontal (0.3s)
- Animações de scroll: elementos se revelam com movimento

**Typography System**:
- Display: Bold sans-serif (ex: IBM Plex Sans Bold) para títulos
- Body: Regular sans-serif (ex: IBM Plex Sans Regular) para textos
- Hierarquia: 3rem (h1), 2rem (h2), 1.5rem (h3), 1rem (body)
- Line-height: 1.5 para corpo, 1.1 para títulos (aperto)

## Design Escolhido: Minimalismo Institucional com Propósito Social

Escolhi a **Resposta 1** por ser a abordagem mais apropriada para o tema. Um site sobre violência doméstica requer confiança, clareza e acessibilidade acima de tudo. O minimalismo institucional garante que a informação seja facilmente compreendida, enquanto o humanismo digital (através de whitespace e tipografia cuidadosa) mantém o aspecto acolhedor e empático necessário para vítimas e apoiadores.

**Paleta Final**:
- Primária: Azul profundo (#1e3a8a)
- Secundária: Bege suave (#f5f1e8)
- Acentos: Vermelho suave (#dc2626) para alertas, Verde suave (#059669) para ações positivas
- Tipografia: Playfair Display (títulos) + Source Sans Pro (corpo)
