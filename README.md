# Domestic Violence Awareness Project

## Description
This project is aimed at raising awareness about domestic violence and providing resources for individuals affected by it. The initiative involves various strategies and partnerships to educate the community and support victims.

## Features
- Comprehensive resources for victims of domestic violence.
- Educational materials for community awareness and prevention.
- Access to local support services and hotlines.
- Community events and workshops.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** Heroku / Netlify

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Periguary/combate-violencia-domestica.git
   ```
2. Navigate to the project directory:
   ```bash
   cd combate-violencia-domestica
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open your browser and go to `http://localhost:3000` to view the project.
- Explore the features and resources provided for domestic violence awareness.
